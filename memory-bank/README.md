# Memory Bank - AI 操作与版本记录

本目录用于记录**每次 AI 对项目的修改**及**版本演进**，便于回溯与保持一致性。

## 文件说明

| 文件 | 用途 |
|------|------|
| [OPERATIONS.md](./OPERATIONS.md) | **操作日志**：每次 AI 修改后追加一条记录，包含日期、版本、摘要、涉及文件、可选说明。 |
| [README.md](./README.md) | 本说明文件。 |

## 使用约定

1. **谁更新**：每次 AI 完成对业务代码或项目文档的修改后，由 AI **自动**在 `OPERATIONS.md` 末尾追加一条操作记录。
2. **何时更新**：在本次对话中完成修改并回复用户之前，必须更新 `OPERATIONS.md`。
3. **记录格式**：见 `OPERATIONS.md` 顶部说明；每条包含日期、版本号、操作摘要、涉及文件、必要时简短说明。
4. **版本号**：按操作顺序递增（如 1, 2, 3…），与日期一起写在每条记录中。

## 与 Cursor 规则的关系

项目 `.cursor/rules` 中已约定：**每次修改完成后必须在 memory-bank/OPERATIONS.md 中更新记录**。因此无需在提问时额外说明「请更新 memory-bank」，AI 会按规则执行。
