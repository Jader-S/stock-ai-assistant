# 项目文档索引

本目录包含 AI 投研助手（stock-ai-assistant）的架构与设计文档，供开发与 AI 协作时查阅。

| 文档 | 内容 |
|------|------|
| [ARCHITECTURE.md](./ARCHITECTURE.md) | 项目结构、技术栈、数据流（聊天 / 存储 / 错误）、环境变量、关键文件职责 |
| [DESIGN.md](./DESIGN.md) | 设计原则、编码规范、UI 与错误约定、修改前检查清单 |

**修改本项目前必须先阅读 [ARCHITECTURE.md](./ARCHITECTURE.md) 与 [DESIGN.md](./DESIGN.md)**，并遵循其中的约定。

AI 每次修改后会在 **[memory-bank/OPERATIONS.md](../memory-bank/OPERATIONS.md)** 中追加操作记录与版本，详见 [memory-bank/README.md](../memory-bank/README.md)。
